# Summary: explain Tree In DSA

* Basic medical terminology covers core concepts, variables, constants, and relationships.
* Binary trees are structures where each node can have a maximum of two children.
* Binary Search Trees (BST) maintain order by placing smaller values on the left and larger values on the right.
* Self-balancing trees, such as AVL and Red-Black trees, are used to maintain structural efficiency.
* Tree traversal techniques include Breadth-First Search (BFS) for level-order searching and Depth-First Search (DFS) for deep exploration.
